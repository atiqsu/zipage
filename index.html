What's Zipage?

The Zipage is an application which helps you to download all/selected images present in the webpage. You need to give the address of the webpage in the box provided.Once the recursive retrieval is complete, you will get a page showing the list of all images present. You can select the images, or select all to zip all of them. It saves the size of download and saves you from the hazzle of downloading single images from the webpage.
